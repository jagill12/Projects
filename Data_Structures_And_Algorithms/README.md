# Overview
This directory in my "Projects" repository is to house the final project I did in my "Data Structures and Algorithms" class. It is a miniature Healthcare Information System I wrote to demonstrate my aptitude with many of the concepts we learned in 
the class, like AVL trees, tree-balancing algorithms, utilizing singly linked lists, the priority queue, dynamic memory management, array-based lists, binary search tree operations, recursion, tree traversal/cleanup, linear search, comparison-based 
ordering, and file-system traversal as a graph/tree walk to name some.
